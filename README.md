# Always On Tecnologia - Site Institucional

Site estático completo para empresa de criação de sistemas web, sites e aplicativos mobile.

## Como configurar o WhatsApp

Abra o arquivo `script.js` e altere:

```js
const WHATSAPP_NUMBER = "5511999999999";
```

Use o formato:
- 55 = Brasil
- 11 = DDD
- 999999999 = número

Exemplo:
```js
const WHATSAPP_NUMBER = "5511987654321";
```

## Como publicar

Você pode publicar em:
- Hostinger
- Locaweb
- Netlify
- Vercel
- GitHub Pages
- IIS/Apache/Nginx

Basta enviar todos os arquivos da pasta para o servidor.
